# Resume Sorting Automation System

This repository provides a **FastAPI** application that serves as the core of the Resume Sorting Automation System. The main entry point (`app.py`) sets up the FastAPI app, configures API‑key authentication, mounts static files for the front‑end UI, and includes routers for handling resume uploads, candidate management, and shortlist operations.

---

## 📄 Code Overview (`app.py`)

```python
import os
from fastapi import FastAPI, Depends, HTTPException, status
from fastapi.security import APIKeyHeader
from fastapi.responses import FileResponse
from fastapi.staticfiles import StaticFiles

from routers import upload, candidates, shortlist

# -------------------------------------------------------------------
# Configuration & Authentication
# -------------------------------------------------------------------
API_KEY = os.getenv("API_KEY", "my-secret-key")
api_key_header = APIKeyHeader(name="X-API-Key", auto_error=False)

def get_api_key(api_key: str = Depends(api_key_header)):
    if api_key == API_KEY:
        return api_key
    raise HTTPException(
        status_code=status.HTTP_403_FORBIDDEN,
        detail="Invalid API Key"
    )

# -------------------------------------------------------------------
# FastAPI Application
# -------------------------------------------------------------------
app = FastAPI(
    title="Resume Sorting Automation System",
    description="API for uploading resumes, extracting candidate data, and managing shortlists."
)

# Serve the front‑end UI (static files located in the `frontend` directory)
app.mount("/static", StaticFiles(directory="frontend"), name="static")

# Root endpoint – returns the main HTML page
@app.get("/", include_in_schema=False)
def root():
    return FileResponse("frontend/index.html")

# -------------------------------------------------------------------
# Router Registration (all routes are protected by the API key)
# -------------------------------------------------------------------
app.include_router(upload.router, prefix="/upload", dependencies=[Depends(get_api_key)])
app.include_router(candidates.router, prefix="", dependencies=[Depends(get_api_key)])
app.include_router(shortlist.router, prefix="", dependencies=[Depends(get_api_key)])
```

## 🚀 Running the Application

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. (Optional) Set a custom API key
# Windows PowerShell
set API_KEY=your‑custom‑key

# 3. Start the server
python -m uvicorn app:app --reload --host 0.0.0.0 --port 8000
```

Open your browser at `http://localhost:8000` to view the UI and start using the API.

## 📚 API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `POST` | `/upload` | Upload one or more resume files (PDF/DOCX). |
| `POST` | `/extract` | Trigger extraction on uploaded resumes. |
| `GET`  | `/candidates` | Retrieve all candidate records (supports pagination). |
| `POST` | `/shortlist/{candidate_id}` | Add a candidate to the shortlist. |
| `GET`  | `/shortlisted` | List all shortlisted candidates. |

All routes require the header `X-API-Key: <your‑api‑key>`.
